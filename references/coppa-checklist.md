# COPPA Checklist (16 CFR Part 312)

FTC children's privacy for apps and AI agents. Verify amended Rule effective dates. Not legal advice.

## Applicability

- [ ] Child-directed service assessment (§312.2)
- [ ] Mixed audience age screen
- [ ] Actual knowledge of under-13 users procedure
- [ ] Operator entity identified

## Notice and consent

- [ ] Direct notice to parents before collection (§312.4)
- [ ] Online privacy policy linked and current
- [ ] Verifiable parental consent before PI collection (§312.5)
- [ ] VPC method documented and auditable
- [ ] Exceptions (internal ops, one-time contact) narrowly applied

## Data practices

- [ ] PI inventory through agent, LLM, MCP, analytics
- [ ] Data minimization—no excess collection in prompts
- [ ] Retention limits and automated deletion (§312.10)
- [ ] Parent review and deletion request process

## Service providers

- [ ] Written LLM/MCP agreements (§312.2 service provider)
- [ ] No independent use or advertising on child PI
- [ ] No model training on child PI without consent
- [ ] Subprocessors disclosed in policy

## Security

- [ ] Reasonable security procedures (§312.8)
- [ ] Encryption, access control, MFA for admin
- [ ] Redaction before LLM where feasible
- [ ] Deanonymize opt-in disabled for child paths

## Restricted practices

- [ ] No conditioning on extra PI disclosure
- [ ] Targeted advertising / ad SDKs reviewed
- [ ] Push notifications and geolocation per amended Rule
- [ ] Agent UX does not solicit unnecessary PI from children

## School + consumer hybrid

- [ ] FERPA school official path documented if school-directed (see ferpa-education-records)
