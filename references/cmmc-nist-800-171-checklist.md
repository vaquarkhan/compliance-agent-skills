# CMMC 2.0 Level 2 / NIST SP 800-171 Checklist

Use with skill `cmmc-nist-800-171`. 110 security practices for CUI.

## Access control (AC)

- [ ] Limit system access to authorized users and processes
- [ ] Limit access to types of transactions and functions
- [ ] Control CUI flow per authorized interfaces
- [ ] Separate duties and least privilege
- [ ] Unsuccessful logon attempts limited
- [ ] Session lock and termination

## Awareness and training (AT)

- [ ] Security awareness for all personnel handling CUI
- [ ] Role-based training for administrators and developers

## Audit and accountability (AU)

- [ ] Audit logs for CUI systems
- [ ] Trace actions to individual users
- [ ] Review and report anomalies

## Configuration management (CM)

- [ ] Baseline configurations and inventory
- [ ] Security configuration settings (STIG/CIS where applicable)
- [ ] Unapproved software controlled

## Identification and authentication (IA)

- [ ] Unique identifiers
- [ ] MFA for network access to CUI environments

## Incident response (IR)

- [ ] IR capability tested
- [ ] Report incidents per DFARS 252.204-7012 (72-hour window)

## SPRS / assessment

- [ ] Self-assessment score calculated and submitted to SPRS
- [ ] POA&M for not-met practices with milestones
- [ ] C3PAO assessment scheduled if required for certification
